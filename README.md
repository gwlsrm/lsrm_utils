# lsrm_utils
python utils for lsrm (http://lsrm.ru/en/) software (spectra, nuclide libraries, etc)

**speparser.py** -- simple parser for '*.spe'-files from Spectraline. Can output header information and convert spectrum to txt-format

**speplotter.py** -- plotter for specta from spe-files. Dependencies: speparser, matplotlib
